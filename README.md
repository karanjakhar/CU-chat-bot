# CU-chat-bot
This is a type of retrival chat bot. It's backend is developed in Flask.     
To run it just download or clone this repository.   
Type following command in terminal :-    
For linux :-  
export FLASK_APP=cubot.py     
python3 -m flask run      

For windows :-   
set FLASK_APP=cubot.py      
python -m flask run
